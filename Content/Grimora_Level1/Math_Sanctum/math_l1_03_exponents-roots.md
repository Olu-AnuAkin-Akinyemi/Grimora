## 1. Lesson Snapshot

### Title  
Exponents & Roots: Powering Up and Revealing the Seed

### One-Sentence Summary  
Exponents are repeated multiplication that “power up” a number, and roots undo that power-up, revealing the original “seed” number.

### Learning Objectives – Learners will be able to…

- Describe an exponent as repeated multiplication.  
- Compute simple powers like \(2^2, 2^3, 3^2\).  
- Explain square root and cube root in simple terms.  
- Recognize exponent and root as inverse operations in basic cases.  
- See how exponential growth connects to scaling in science and engineering.

### Key Terms

- **Exponent** – The small raised number that tells how many times to multiply the base by itself.  
- **Base** – The number being repeatedly multiplied.  
- **Power** – The result of an exponent expression (e.g., \(2^3\)).  
- **Square root** – A number that, when multiplied by itself, gives the original number.  
- **Cube root** – A number that, when multiplied by itself three times, gives the original number.

> [Deeper Study – From Linear to Exponential]  
> For older learners, contrast “add 2 each step” (linear) with  
> “multiply by 2 each step” (exponential). Show small table of values  
> to see how quickly exponentials grow.

## 2. Narrative Hook

In a Kimia lab, the master says:

- “This mixture **doubles** in strength each time we repeat the heating ritual.”  
  - 1st heating → strength ×2  
  - 2nd heating → ×4  
  - 3rd heating → ×8  

Instead of writing ×2, ×2, ×2 again and again, they use an **exponent** to track the mixture’s **power level**. Later, when they need to calm the mixture down, they use a **root** to figure out what base strength they started from.

**Why This Matters (Now & Later)**

- **Now:**  
  - Makes it easier to talk about repeated multiplication without writing long chains.  
  - Shows why some things grow faster than just “add the same amount.”
- **Later (science & engineering):**  
  - Signals, sound, light, and population can change exponentially.  
  - Roots show up when solving for original values (e.g., side length of a square from its area).

## 4. Main Concept Explanation

### 4.1 Intuitive Explanation

- \(2^3\) means:
  - Base = 2  
  - Exponent = 3  
  - “Multiply 2 by itself 3 times” → \(2 × 2 × 2 = 8\).

Visuals:

- Arrays / blocks:
  - \(2^2\): 2 by 2 square of blocks.  
  - \(3^2\): 3 by 3 square.  
- Stacked levels:
  - Think of each multiplication as a new “layer” of power.

> [Deeper Study – Geometry Link]  
> Show that “squared” connects to the area of a square (side × side)  
> and “cubed” to volume of a cube (side × side × side).

### 4.2 Formal / Semi-Formal Explanation

- General form:
  - \(a^n = a × a × \dots × a\) (with `a` repeated `n` times).  

- Special cases:
  - \(a^1 = a\)  
  - \(a^2\): “a squared”  
  - \(a^3\): “a cubed”

- Roots:
  - Square root: \(\sqrt{9} = 3\) because \(3^2 = 9\).  
  - Cube root: \(\sqrt[3]{8} = 2\) because \(2^3 = 8\).

- Inverse relationship:
  - Exponent and matching root undo each other in basic cases.  
  - If \(a^2 = b\), then \(\sqrt{b} = a\).  
  - If \(a^3 = c\), then \(\sqrt[3]{c} = a\).

### 4.3 Worked Examples

- Powers:
  - \(2^2 = 4\), \(2^3 = 8\), \(2^4 = 16\).  
  - \(3^2 = 9\), \(4^2 = 16\), \(5^2 = 25\).

- Roots (perfect squares):
  - \(\sqrt{4} = 2\), \(\sqrt{9} = 3\), \(\sqrt{16} = 4\), \(\sqrt{25} = 5\).

- Roots (perfect cubes):
  - \(\sqrt[3]{8} = 2\), \(\sqrt[3]{27} = 3\).

## 5. Guided Practice (with hints)

### 5.1 Practice Set (4–8 items)

Examples:

1. Compute: \(2^3 = ?\)  
   - Hint: Multiply 2 by itself 3 times.

2. Compute: \(3^2 = ?\)  
   - Hint: 3 × 3.

3. Compute: \(5^2 = ?\)  
   - Hint: Think “5 times itself.”

4. Which exponent gives 64?  
   - \(2^5\) or \(2^6\)?  
   - Hint: List powers: \(2^3 = 8\), \(2^4 = 16\), \(2^5 = 32\), \(2^6 = ?\).

5. Fill in: \(\sqrt{25} = ?\)  
   - Hint: “What number times itself gives 25?”

### 6.1 Practice Set (8–10 items)

- **Word problems (1–2 items):**
  1. “A small robot doubles the number of scanned tiles each step.  
     It starts with 1 tile in step 0. After 3 steps, how many tiles has it scanned in that step pattern (ignoring earlier steps)?”

  2. “A square garden has area 25 square meters. Each side has the same length.  
     What is the length of one side?”  
     - Hint: Think \(\sqrt{25}\).

  - “Explain in your own words how a square root ‘undoes’ a square.”

## 9. Mini Quiz

### 9.1 Quiz (3–5 questions)

1. Compute: \(3^2 = ?\)  
2. Compute: \(2^4 = ?\)  
3. Fill in: \(\sqrt{16} = ?\)  
4. Which is larger: \(2^3\) or \(3^2\)?  
5. In one sentence: “What does the square root of a number tell you?”

### 9.2 Answer Key

1. \(3^2 = 9\)  
2. \(2^4 = 16\)  
3. \(\sqrt{16} = 4\)  
4. \(2^3 = 8\), \(3^2 = 9\) → \(3^2\) is larger.  
5. Sample idea: “The square root tells you what number, multiplied by itself, gives the original number.”
